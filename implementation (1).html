OpenCV-Based Models for Sandalwood Spike Disease (SSD) Detection

PHYSICAL DETECTION EXPLANATION

Leaf & Branch Features

  • Leaf Yellowing → Detects color changes from healthy green to yellow/brown.
  • Leaf Size Reduction → Measure shape and size shrinkage.
  • Leaf Curling → Identify irregular edge contours.
  • Branch Witches' Brooming → Spot unnatural bushy, clustered branch growth.

Color Analysis

Convert images to HSV (Hue, Saturation, Value) and extract hue range differences.

  • Healthy leaves: (H: 30-90, S: 40-100, V: 50-255)
  • Infected leaves: (H: 10-40, S: 10-80, V: 30-200)

Texture & Shape Features

  • Edge Detection → Sobel/Canny filters for vein analysis.
  • Blob Detection → Identify clustered small leaves.
  • Contour Analysis → Detects irregular leaf margins.

Technical Modelling

Feature Extraction

To detect SSD based on physical symptoms like witch’s broom branching and leaf discoloration.

Color Segmentation:

  • Convert images to HSV color space.
  • Identify yellowing or reddish discoloration in leaves.
  • cv2.inRange() to isolate affected regions.

Edge Detection & Shape Analysis:

  • cv2.Canny() to detect sharp, abnormal branching.
  • cv2.findContours() to analyze leaf and branch deformation.

Texture Analysis:

  • Use GLCM (Gray-Level Co-occurrence Matrix) or LBP (Local Binary Pattern) for feature extraction.
  • Apply a Support Vector Machine (SVM) classifier for detection.

Deep Learning

Deep learning model with OpenCV for real-time detection.

  • MobileNetV2 (Lightweight, good for drones)
  • YOLOv8 (Best for real-time object detection)

Hyperspectral Imaging

Uses infrared cameras to detect chlorophyll degradation. Only usable if integrated with a multispectral camera on an agri-drone.

Hardware

Bioreactor

Our bioreactor is a lab-scale stirred-tank model tailored for the efficient growth of Bacillus subtilis strain 168. The design ensures optimal conditions for microbial proliferation, focusing on precise control of environmental parameters to maximize biomass yield and metabolic activity [1],[2].

  • Tank Design: Cylindrical, autoclavable vessel made from stainless steel or borosilicate glass. Working volume: 5L, ideal for small-scale experiments.
  • Stirring Mechanism: Equipped with a Rushton turbine impeller for efficient mixing and oxygen transfer. Adjustable stirring speed to prevent cell shear stress while maintaining homogeneity.
  • Aeration System: Sparger system at the vessel base introduces sterile air, ensuring high dissolved oxygen levels. Dissolved Oxygen (DO) probe for real-time monitoring to maintain aerobic conditions.
  • Environmental Control: pH control within the optimal range of 6.5 to 7.5 using automatic acid/base addition. Temperature regulation is maintained at 30–35°C through a water jacket or heating mantle. High oxygen environment with aeration rate optimized at 1 vvm (volume of air per volume of liquid per minute).
  • Sampling & Monitoring: Sterile sampling ports for periodic collection of culture. Integration of temperature, pH, and DO sensors connected to a digital logging system.

Modifications to AgriDrone

  • Connections:
    • A] Pump → Relay Module → Flight Controller AUX Pin (PWM Output)
    • B] Solenoid Valve → Relay Module → Flight Controller AUX Pin
    • C] Flow Sensor → ADC Pin of Flight Controller
  • Embedded Software (Drone Firmware & Controllers):
    • Modify the flight controller (PX4, Ardupilot) to handle the spray system
    • Implement PID control for optimal altitude adjustment during spraying
    • Integrate solenoid valve control for real-time solution delivery
  • Database & IoT Integration:
    • Cloud-based PostgreSQL database for tracking affected trees
    • Edge computing via ESP32/Raspberry Pi for real-time processing
    • Communication via LoRa or 4G LTE for low-latency updates
  • Component Specifications:
    • A] Reservoir System (Lightweight & Compact):
      • Capacity: 1L
      • Component: Collapsible Soft Tank
      • Material: Polyethylene (PE) or TPU
      • Weight: 200 g (empty), 1.2 kg (full)
    • B] Spray Pump & Pressure System:
      • Component: Miniature Electric Diaphragm Pump
      • Model: 12V DC Diaphragm Pump (~3-5L/min flow rate)
      • Pressure: 30-60 PSI (sufficient for foliar spray)
      • Weight: ~500g - 800g
      • Example Product: Shurflo 2088-554-144 Agricultural Diaphragm, Seaflo 12V 4.5L/min Pump
    • C] Spray Nozzle System:
      • Component: Adjustable Foliar Spray Nozzle (Fine Mist Type)
      • Nozzle Type: Venturi/Flat Fan/Full Cone Nozzle
      • Droplet Size: ~150-200 microns (for effective plant absorption)
      • Weight: ~100g-200g
      • Example Product: Lechler 110-degree Agricultural Spray, TeeJet AIXR11002VP Flat Fan Nozzle
    • D] Flow Control & Valve System:
      • Component: Solenoid Valve (for controlling flow rate)
      • Voltage: 12V DC
      • Type: Normally Closed (NC)
      • Weight: ~250g
      • Example Product: Bürkert 12V DC Solenoid Valve 2/2-way
    • E] Flow Sensors & IoT Monitoring:
      • Component: Digital Flow Meter & Pressure Sensor
      • Sensor Type: Hall Effect Flow Sensor (YF-S201 / FS300A)
      • Flow Rate: 0.3 – 6L/min
      • Weight: ~100g
      • Example Product: G1/4 YF-S201 Water Flow Sensor, FS300A Agricultural Flow Sensor
    • F] IoT Integration Hardware (Real-Time Monitoring):
      • Component: ESP32 / Raspberry Pi (for wireless data transmission)
      • Microcontroller: ESP32 (low power & lightweight)
      • Connectivity: WiFi / LoRa (for long-range communication with database)
      • Weight: ESP32 (~50g), Raspberry Pi (~100g)
      • Example Product: ESP32 WROOM32 IoT Board, Heltec LoRa ESP32 (for long-range data collection)

Delivery

These studies were read through to confirm the effectiveness of Bacillus subtilis as a foliar spray to improve plant growth and control diseases in various crops. Foliar application of Bacillus subtilis OSU-142 on apricot trees enhanced growth, yield, and helped control shot-hole disease (Coryneum blight) [3]. Another study highlighted the effectiveness of Bacillus subtilis in protecting plants from fungal diseases, demonstrating its role as a biocontrol agent [4]. Additionally, Bacillus subtilis was shown to produce antimicrobial substances that suppress harmful bacteria and fungi, contributing to plant health [5]. In rice cultivation, the combined foliar spraying of Bacillus subtilis and Achromobacter xylosoxidans effectively managed bacterial leaf blight under field conditions [6]. Together, these studies demonstrate that Bacillus subtilis can be an effective, natural method to enhance crop health and disease resistance.
